# Generator-C — генератор случайных цитат

Generator-C — учебный проект на Python, представляющий собой генератор случайных цитат.
Приложение поддерживает работу в режиме командной строки и через веб-интерфейс.

Проект создан с целью практики работы с Python, Flask, структурой проекта,
системой контроля версий Git и CI (GitHub Actions).

---

## Возможности

- получение случайной цитаты через CLI
- веб-страница с кнопкой генерации новой цитаты
- API endpoint `/random`, возвращающий цитату в формате JSON
- автоматическая проверка кода при каждом push
- контроль стиля кода с помощью flake8
